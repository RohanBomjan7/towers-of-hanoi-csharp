# Towers of Hanoi - C#

A C# implementation of the classic Towers of Hanoi problem, built as part of CIS 300 at Kansas State University.

## About the Project

The Towers of Hanoi is a classic mathematical puzzle where the goal is to move a stack of disks from one peg to another, following these rules:
- Only one disk can be moved at a time
- A disk can only be placed on top of a larger disk
- All disks must be moved from the source peg to the destination peg

This project implements the solution using **recursion** in C# with a Windows Forms user interface.
